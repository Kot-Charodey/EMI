# EMI
Сетевая библиотека, реализующая принцип удаленных процедур. Универсальная библиотека для быстрой и эффективной передачи пакетов, с модульной структурой, с низкоуровневым использованием системного API. 
Проект возник из-за необходимости быстро и удобно передавать большие объемы данных, синхронизировать состояния.
Могу продемонстрировать примеры использования в других личных проектах
